# Starling · 椋鸟

一个纯前端的小工具集合，平时自己整理扫描件 / OCR 文本时顺手写的，凑合能用，分享出来给需要的朋友。

没有后端，没有账号，所有处理都在你自己浏览器里跑，文件不会被传到任何地方。

## 能做什么

- **PDF 调页**：逐页调亮度、对比度、Gamma、黑白场、锐化，还能自动检测并校正倾斜角度，处理完导出一份新 PDF。不影响原始分辨率。支持连续多步撤销（拖滑块、点确认、点重置都算一步），也可以直接 `Ctrl+Z`。
- **换行清理**：贴一段 OCR 识别出来的文本，一键去掉不该有的换行，只保留真正的自然段。
- **脚注转换**：正文里的 abcd 角标 + 对应的注释列表，一键转成带层级编号的 Markdown 脚注格式。

三个功能各自独立，互不影响，页签切换即可。

## 怎么用

直接用浏览器打开 `index.html` 就行（有的浏览器对 `file://` 打开的 ES module 有限制，如果 PDF 调页那页空白，起个本地静态服务器更稳：

```bash
python -m http.server 8080
# 然后浏览器打开 http://127.0.0.1:8080
```

支持安装成 PWA（地址栏或浏览器菜单里的"安装"选项），装完之后大部分离线也能用。

## 技术上没什么特别的

- 没有构建流程，纯手写 HTML/CSS/JS，改完直接刷新看效果
- PDF 相关功能依赖 [pdf.js](https://mozilla.github.io/pdf.js/) 和 [pdf-lib](https://pdf-lib.js.org/)，已经下载放进 `vendor/` 目录里本地引用，不连 CDN
- `sw.js` 是 Service Worker，负责离线缓存
